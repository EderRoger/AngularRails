AngularRails
============

AngularJS consuming rails server app

How to install

Requirements

Rails 4
PostgreSQL

clone the project

`git clone git@github.com:EderRoger/AngularRails.git`

`cd AngularRails`

`bundle install`

`rake db:create`

`rake db:migrate`

`rails s`

`http://localhost:3000/#/pessoas`
